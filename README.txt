# &WebLab — Starter
Sito starter professionale su sfondo verde con testi neri in grassetto.

## Come usarlo
1. Carica `index.html` e `style.css` nella **root** del tuo repo GitHub Pages (`giuppe5401-stack.github.io`).
2. In **Settings → Pages** imposta **Branch: main** e **Folder: /(root)**.
3. Apri `https://giuppe5401-stack.github.io/` e aggiorna (Ctrl+F5).

## Cosa personalizzare (cerca [TODO] in `index.html`)
- Hero: titolo e sottotitolo.
- Servizi: descrizioni brevi.
- Portfolio: sostituisci i placeholder coi tuoi progetti.
- Footer: inserisci la **tua email**.
- WhatsApp: il link usa +39 347 944 1079; cambialo se serve.

## Problemi comuni
- Se non vedi nulla, usa il metodo `/docs`: sposta i file in una cartella `docs` e in **Settings → Pages** seleziona `/(docs)`.
- Forza la cache con Ctrl+F5 o apri in finestra in incognito.
